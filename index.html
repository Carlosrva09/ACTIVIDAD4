ELEMENTOS DE UN CPD

ELEMENTO CARACTERÍSTICAS ESTÁNDARES
Climatización Control de temperatura y humedad para garantizar condiciones óptimas para equipos electrónicos. Temperatura: 18-27 °C, Humedad: 40-60%
Energía eléctrica Suministro eléctrico ininterrumpido con sistemas de respaldo como UPS y generadores. ANSI/TIA-942, Uptime Institute Tier Standard
Cableado Cableado estructurado para redes de datos y voz, con capacidad para futuras expansiones. ANSI/TIA-568, ISO/IEC 11801
Seguridad fisíca Acceso restringido mediante sistemas de identificación y registro de visitantes, vigilancia por video, etc. ISO/IEC 27001, SSAE 18 / ISAE 3402
Protección contra incendios Sistemas de detección y extinción de incendios, como rociadores y extintores. NFPA 75, NFPA 76, ISO 14520
Sistemas de respaldo UPS (Sistema de Alimentación Ininterrumpida) y generadores para asegurar la continuidad del servicio. NEMA SG 5, IEC 62040-3
Monitoreo Sistemas de monitoreo remoto para supervisar el funcionamiento de los equipos y detectar problemas ISO/IEC 27001, ANSI/TIA-942
Conectividad Conexiones redundantes a proveedores de servicios de Internet y redes privadas para garantizar la conectividad. ANSI/TIA-942, ISO/IEC 27001
